se o usuário clicar em "fazer análise" fechará a tela atual e irá abrir uma tela com um local para pesquisar dentro dos arquvios do próprio computador, estará escrito para este: "Entre com o arquivo .csv"

em baixo dessa tela terá um botão escrito próximo "passo", que levará o usuário para uma tela final onde terá botões: "Análise 1", "Analise 2", "Analise 3"

entretanto, se o usuário tiver selecionado "corte por botão do 2 step(Arrancada)" na tela anterior que havia checkbox, já levará o usuário direto para tela com os botões: "Análise 1", "Analise 2", "Analise 3"

se o usuário tiver selecionado "corte por distância" na tela anterior que havia checkbox, já levará o usuário direto para tela com os botões: "Análise 1", "Analise 2", "Analise 3"
